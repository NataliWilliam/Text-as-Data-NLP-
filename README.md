## üß™ Methods

This project uses basic Natural Language Processing (NLP) techniques
including: - Text preprocessing and cleaning - Tokenization and
lemmatization - Stopword removal - Readability metrics (Flesch Reading
Ease and Flesch-Kincaid Grade Level) - Word frequency analysis and word
clouds - Group comparisons by gender and education level

Education was categorized into: - Advanced - Not Advanced

Gender categories: - Male - Female

## üìà Results

Key findings and interpretations are discussed in detail in the project

The analysis focuses on: - Essay length - Readability - Common word
usage - Differences across demographic groups
